AudioCascade 1.0.0
===================

Smart Audio Device Manager for macOS

Installation:
1. Open AudioCascade-1.0.0.dmg
2. Drag AudioCascade to Applications
3. Eject the DMG
4. Launch AudioCascade from Applications
5. Grant accessibility permissions when prompted

First Launch:
- Right-click AudioCascade and select "Open"
- Click "Open" in the security dialog
- This is only needed once

Features:
- Priority-based audio device switching
- Keyboard shortcuts for instant switching
- Remembers all your audio devices
- Beautiful native macOS interface
- Supports multiple languages (EN/DE/FR)

Requirements:
- macOS 12.0 or later
- Accessibility permissions (for keyboard shortcuts)

Support:
- GitHub: https://github.com/Cosnavel/AudioCascade

Enjoy AudioCascade!
